ESP32 Smart Watch

Advanced Embedded Systems & IoT Integrated Wearable Device

Project Overview

This project is a fully functional smart watch built using the ESP32 microcontroller. It is designed not only as a wearable device but as a comprehensive platform for learning embedded systems, real-time data acquisition, and IoT communication.

Key Objective: Build a compact, multi-sensor embedded system capable of real-time monitoring, wireless data transmission, and user interaction.

The system integrates multiple sensors, a display interface, wireless connectivity, and a backend server to demonstrate end-to-end IoT architecture.

Environmental Monitoring (Temperature & Pressure)

The device uses environmental sensors (such as BME280) to measure temperature and atmospheric pressure. These measurements are essential for applications like weather monitoring and altitude estimation.

Engineering Insight: Atmospheric pressure data can be used to estimate altitude changes, making this system extendable for outdoor and navigation-based applications.

Humidity Sensing

Along with temperature and pressure, the system measures humidity, enabling complete environmental analysis. This makes the device suitable for climate monitoring and smart environment applications.

IoT Data Transmission

The ESP32 transmits collected sensor data over WiFi to a remote server. This demonstrates real-time IoT communication and remote monitoring capabilities.

Architecture: Sensor → ESP32 → WiFi → Web Server → User Interface

Heart Rate Monitoring System

The smart watch integrates a pulse sensor to measure heart rate in real time. This feature transforms the device into a basic health monitoring system.

Working Principle: The pulse sensor detects blood flow variations using light absorption, which are processed by ESP32 to calculate BPM (Beats Per Minute).

Digital Compass

The system includes a magnetometer-based digital compass for directional sensing. Proper calibration is required to eliminate magnetic interference and improve accuracy.

Note: Calibration is crucial for accurate heading calculation due to environmental magnetic distortions.

Date & Time Display

The smart watch also functions as a standard timekeeping device by displaying the current date. This feature enhances usability as a daily wearable gadget.

Hardware Components

The system is built using ESP32 as the main controller along with sensors like BME280. These components enable wireless communication and multi-parameter sensing.

Core Components: ESP32, BME280 Sensor, Pulse Sensor, Display Module

User Settings Interface

The device includes an onboard settings interface that allows users to configure system parameters. This improves flexibility and usability.

Web-Based Configuration

In addition to onboard controls, the system provides a web interface for remote configuration. This demonstrates full-stack IoT system design.
